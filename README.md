# FAOSTAT-agrotoxin-data
This script analyzes data from [FAOSTAT](https://www.fao.org/faostat/en/#home). It is divided into three main sections: 

1. **Configuration**;
2. **Data Wrangling**;
3. **Graph Creation**. 
# How it Works

The Configuration section allows you to change the analysis by setting various parameters. The Data Wrangling section cleans and prepares the data for visualization. And, lastly, the Graph Creation section creates a bar graph, helping visualize the top countries based on the specified criteria. This script is designed to be flexible and adaptable to any FAOSTAT data (hopefully), making it a useful tool for exploring and visualizing FAOSTAT datasets.

# Requirements

It should all be automatically downloaded and installed, but if it isn't, the libraries used where:

1. here (for file path management and organization)
2. tidyverse (for data manipulation and visualization)
3. rmarkdown (for saving the file in a different place)
